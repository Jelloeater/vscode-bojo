### Added

- Initial release of Bojo — Bullet Journal Task Cycler
- Forward cycle command (`Ctrl+Enter` / `Cmd+Enter`)
- Reverse cycle command (`Ctrl+Shift+Enter` / `Cmd+Shift+Enter`)
- Support for standard markdown list markers (`-`, `*`, `+`)
- Multi-cursor support for cycling multiple tasks at once
- Keybindings scoped to markdown files only
- Task state cycle: `[ ]` → `[/]` → `[!]` → `[x]` → `[ ]`
